# winforms-calculator
C# Windows Forms kullanılarak geliştirilmiş hesap makinesi uygulaması.
## Özellikler
- Toplama, çıkarma, çarpma, bölme
- Ondalıklı (virgüllü) sayılar
- Temizleme (C) işlemi
- Windows Forms arayüzü

## Kullanılan Teknolojiler
- C#
- Windows Forms
- .NET Framework

## Amaç
Bu proje, Visual Programming / C# Windows Forms dersi kapsamında
temel olay yönetimi (Event Handling) ve arayüz tasarımını öğrenmek amacıyla geliştirilmiştir.
